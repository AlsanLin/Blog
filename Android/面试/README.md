## 学习建议

1. **协程/Flow > Compose > Clean Architecture** 是面试和工作中绕不开的硬通货，优先吃透
2. **KMP ≈ TFLite** 是差异化竞争力，有余力时再补
3. 每个模块都给出了学习路径，跟着走比漫无目的地刷题高效得多
4. 答题时不要只背结论，要能说出 **为什么** 和 **怎么选**
